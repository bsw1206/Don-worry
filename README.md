## 🤝 협업 가이드 및 주의사항 (Ground Rules)

우리 팀의 지속적인 성장과 효율적인 개발을 위해 아래 사항들을 반드시 준수합니다.

### 1. 코드 규칙 
* **커밋 메시지 규칙 준수**: 변경 사항을 명확히 알 수 있도록 정해진 커밋 컨벤션(예: `feat:`, `fix:`, `docs:`)을 준수합니다.

### 2. 데이터 파이프라인 관리 주의사항
* **데이터 일관성 유지**: 금융 API 데이터 수집 및 전처리 과정에서 데이터 유실이나 중복이 발생하지 않도록 정제 규칙을 엄격히 적용합니다.
* **환경 변수 보안**: API Key, DB 계정 정보 등 민감한 설정 값은 절대 Git 저장소에 업로드하지 않으며, `.env` 파일을 통해 안전하게 관리합니다.

### 3. 소통 및 갈등 해결
* **투명한 공유**: 기획 변경이나 기술적 이슈가 발생하면 즉시 공유하고, 대화와 토론을 통해 함께 최선의 해결책을 모색합니다.
